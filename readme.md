# Dashboard E-commerce

## Pasos
    - Crear entorno virtual:
        - python -m venv venv
    - Activar entorno virtual:
        - venv\Scripts\activate
    - Crear archivo requirements.txt:
        - pip freeze > requirements.txt
    - Instalar dependencias:
        - pip install -r requirements.txt
    - Crear archivo .env:
        - touch .env
